# Cowboys AC Tools (PWA)

This is the Progressive Web App for Cowboys AC field operations.
- Installable on iPads (Add to Home Screen)
- Works offline using service worker caching
- Modules include warranty submission, installer checklist, parts restock, tech mapouts, and more.

## How to Deploy
1. Upload files to GitHub repo (public or private)
2. Enable GitHub Pages (Settings > Pages > Source: `main`, Folder: `/root`)
3. Ensure `.nojekyll` is in the root to disable Jekyll processing
4. Visit the published URL and test on your iPad
